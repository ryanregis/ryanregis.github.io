BuzzFood

BuzzFood  is a mock online magazine of BuzzFeed.com
But instead of tackling trendy modern general topics, BuzzFood  represents current events about the food world and some popular delicious recipes.

This responsive online magazine is created by Ryan Gerome Regis as the first mini-project for the Batch 3 of KodeGo Web Development Bootcamp.

Hi! My name is Ryan Gerome Regis, I graduated as a Bachelor of Science in Electronics Engineering Student at UE Caloocan back in 2020. My main focus was in embedded software engineering which is both slightly different and the same from Web Developing.
